# Pilots

STUF pilots should focus on environments where update trust is important, visible, and testable.

## Candidate pilot areas

- fleet operations
- logistics
- municipal services
- industrial IoT
- energy field operations
- connected vehicle support systems
- embedded product update flows

## Pilot goals

- signed metadata verification
- artifact hash and length checks
- staged update authorization
- rollback and freeze protection
- audit logs
- constrained-device feasibility
