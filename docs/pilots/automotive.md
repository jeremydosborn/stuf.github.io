# Automotive primitives

STUF is not currently a full Uptane implementation.

The near-term goal is to explore automotive-adjacent security primitives that may be useful for connected fleets and staged update delivery.

## Candidate primitives

- staged metadata mirrors
- target authorization checks
- fleet-specific policy layers
- constrained verifier profiles
- audit logs for rollout decisions

## Long-term direction

A future profile may explore deeper Uptane compatibility, but the first milestone is practical verification infrastructure.
