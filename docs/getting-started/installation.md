# Installation

STUF is a Rust project.

## Requirements

- Rust toolchain
- Cargo
- Git

## Clone

    git clone https://github.com/jeremydosborn/stuf.git
    cd stuf

## Check the toolchain

    rustc --version
    cargo --version

## Run tests

    cargo test

## Optional embedded target

    rustup target add thumbv7m-none-eabi
